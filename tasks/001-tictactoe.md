# Task: Tic-Tac-Toe

## Goal
Build a simple tic-tac-toe game in a single HTML file.

## Requirements
- Two players take turns clicking cells
- X and O alternate
- Detect win/draw
- Reset button
- runnable with npm run dev

## Constraints
- Single HTML file only
- No external dependencies

## Success Criteria
- Game is playable
- Win detection works
- Reset works
