# Task 06: Canvas UI - Board & Pieces Rendering

## Description
Implement the canvas-based UI: draw the board, pieces using Unicode symbols, and basic styling.

## Definition of Done
- Canvas renders 8×8 grid with alternating light/dark square colors (modern palette)
- Pieces rendered as Unicode chess symbols (♔♕♖♗♘♙♚♛♜♝♞♟) centered in squares
- Board is drawn with a border/frame
- Canvas uses a fixed aspect ratio and scales responsively to window size
- Highlight selected square with a colored border/overlay
- Show valid move indicators (colored dots) for selected piece
