# Task 01: Project Setup

## Description
Initialize the project with package.json, Vite dev server, and the basic file structure.

## Definition of Done
- `package.json` exists with `npm run dev` script using Vite
- `index.html` loads the app
- `src/` directory with `main.js`, `chess.js`, `ai.js`, `ui.js`
- Running `npm run dev` starts the dev server and shows a blank page
