# Task 10: Pawn Promotion UI Overlay

## Description
Implement a UI overlay on the canvas for choosing promotion piece.

## Definition of Done
- When a pawn reaches the last rank, the game pauses and shows a promotion selector
- Overlay shows 4 piece options: Queen, Rook, Bishop, Knight
- Clicking an option completes the move with the chosen promotion
- Overlay is rendered on the canvas (semi-transparent background)
- Works for both human (White) and AI (Black) promotions (AI auto-chooses Queen)
