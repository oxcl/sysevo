# Task 07: Canvas UI - Mouse/Touch Interactions

## Description
Implement click-to-move and drag-to-move interactions on the canvas.

## Definition of Done
- Click-to-move: click a piece to select it, click a valid square to move
- Drag-to-move: mousedown on piece, drag to target square, mouseup to move
- Clicking an opponent's piece that is a valid capture target also works
- Clicking an empty valid square moves the piece there
- Clicking another own piece switches selection
- Clicking invalid square deselects
- Touch events supported (touchstart/touchmove/touchend)
